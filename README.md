# E-Commerce System

A full-stack e-commerce platform designed with a **domain-first approach**, focusing on clear system design, scalable architecture, and clean backend implementation.

The project includes domain modeling, system flows, API design, and architectural documentation before implementation.

## Tech Stack

### Backend

* **ASP.NET Core (C#)**
* **Entity Framework Core**
* **PostgreSQL**
* **JWT Authentication**

### Frontend

* **React**
* **TypeScript**
* **Vite**

### Payments

* **PIX**
* **Bitcoin Lightning Network**

## Documentation

The project documentation is located in the `docs` directory and includes:

* **Domain Model**
  System entities and relationships.

* **System Flows**
  Functional flows for both customers and administrators.

* **API Design**
  REST API specification including request and response structures.

* **Architecture**
  High-level system architecture and component responsibilities.
